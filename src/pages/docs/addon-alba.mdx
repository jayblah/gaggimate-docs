### Mounting: Tape

On non magnetic machines or those with a simple plastic cover over the tank (for example Rancilio Silvia), the included VHB tape can be used to mount the sensor.

<CB>Position the VHB dot on the back of the casing and stick it on. Make sure to apply pressure for 10 seconds for secure adhesion.</CB>

The settings allow you to set the regular idle color / brightness of the LED.
The numbers can be set from 0-255 and control the separate channels of the LED where 0 is off and 255 is full brightness.

The external LED setting allows you to control the brightness of an externally connected LED (See section 6).

For the tank level to work properly you have to configure the distances from the sensor down to 2 points of the tank. See the diagram next to it for an explanation.
You can also use the buttons right of the input fields to set the current value of the sensor.

## 6. External LED / Barista Lights

The Alba PCB allows connecting one or multiple external LEDs to your machine. These can for example be used for Barista lights or other additional lighting.

You can use 5V LEDs with up to 200 mA current in total (1W at 5V). If you exceed this limit you risk damaging the Alba PCB. The connected LEDs can be dimmed to your preferred brightness.
