## Contents

## 1. Preparation

**Safety First!**
Unplug your machine to ensure there's no live connections while working on it.
Work in a dry environment to prevent electrical hazards.

Start by gathering all the materials and tools listed in the sourcing pages.
Follow the Gaggimate installation guides if you haven't already: [Flashing](/docs/flashing).

Read through the entire guide first to ensure any questions will be cleared before the installation. You can always contact the Discord for help.

#### Check the screen

<Preview src={ribbon} width={400} alt="Screen ribbon" />

Before assembling the screen, check that the ribbon is seated and locked correctly. Unlock the connector by pulling down the black locking tab, insert the cable, and close it back up.

## 4. Machine Preparation

Disassemble the machine: remove water lid, drip tray, portafilter, and cup warmer. Disconnect grounding wires.
Familiarize yourself with machine layout and label all wires for reference.

Note that some steps differ from the Gaggimate installation documentation. Please join Discord if you have questions. Once I have more information, I’ll add those specific steps to this document.
